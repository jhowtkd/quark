# Phase 10 Plan: Structured Report Format

## Goal
Enforce a disciplined due-diligence report structure for the economia profile: Tese Principal, Evidencias Verificadas, Fragilidades e Riscos, Premissas Explicitas, Cenarios (Bear/Base/Bull with probabilities and triggers). Each section must contain only relevant content type.

## Must Haves
- [ ] FORMAT-01: Reports follow fixed section order: Tese, Evidencias, Fragilidades, Premissas, Cenarios
- [ ] FORMAT-02: Each section contains only content relevant to its heading
- [ ] FORMAT-03: Quantified tables accompany narrative in Evidencias and Cenarios
- [ ] FORMAT-04: Scenarios include explicit probability weights and trigger conditions
- [ ] All existing functionality preserved
- [ ] Python syntax valid on all modified files
