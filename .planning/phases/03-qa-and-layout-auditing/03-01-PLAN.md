# Plan 03-01 — Build and Authentication Flow Verification

## Scope
Verify the authentication flows (login and register) remain fully functional after Phases 1–2 styling changes. Run the build gate, inspect auth pages for CSS regressions, and confirm form submission logic is intact.

## Must Haves (derived from phase goal)
1. `npm run build` exits with code 0 and produces no TypeScript/Vue syntax errors.
2. Login and register forms use the Blueprint Noir monochrome contract: 0px radius, bottom-border inputs, primary black buttons with block-shadow hover.
3. Form validation, API calls, and routing logic remain unmodified and functional.
