# Plan 02-03: Card and Panel Styling Refactor

## Goal
Update all card and panel surfaces to the Blueprint Noir tonal layering contract: `#e2e2e2` backgrounds, 0px radius, no borders (tonal shift for depth), and hover block-shadow with translate. Also refactor modal panels to use the same flat, sharp aesthetic.

## must_haves
- All cards must have `border-radius: 0px` and `background: #e2e2e2`.
- Cards must have no border (`border: none`) — depth comes from tonal contrast against `#f9f9f9` page background.
- Card hover must be `box-shadow: 4px 4px 0 #000000; transform: translate(-2px, -2px);`.
- Modal panels must have `border-radius: 0px`, flat backgrounds (`#e2e2e2` or `#f9f9f9`), and no drop shadows.
- History database status dots and progress text must use monochrome tones only (`#000000`, `#5e5e5e`, `#c6c6c6`).
