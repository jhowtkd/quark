# Plan 02-02: Input Styling Refactor

## Goal
Convert all form inputs, chat inputs, survey inputs, and code inputs to the Blueprint Noir bottom-border-only contract: transparent background, 0px radius, 2px solid bottom border (`#777777` default, `#000000` on focus, `#ba1a1a` on error), and Work Sans typography.

## must_haves
- All inputs must have `border-radius: 0px`.
- Background must be `transparent`.
- Only the bottom border remains: `border-bottom: 2px solid #777777`; all other borders are `none`.
- Focus state changes bottom border to `#000000` with `outline: none`.
- Error state changes bottom border to `#ba1a1a`.
- Inputs must use `font-family: var(--font-human)` (Work Sans) at 16px, weight 400.
