# Plan 02-01: Button Styling Refactor

## Goal
Update every button class across the application to match the Blueprint Noir analog brutalism contract: 0px border radius, primary/secondary fills, inversion hover with `box-shadow: 4px 4px 0 #000000`, and focus outlines.

## must_haves
- All buttons must have `border-radius: 0px`.
- Primary buttons use `background: #000000; color: #e2e2e2;` and invert on hover.
- Secondary buttons use `background: #f9f9f9; color: #000000; border: 2px solid #000000;` and invert on hover.
- Hover state must include `box-shadow: 4px 4px 0 #000000`.
- Disabled state must be `opacity: 0.4; cursor: not-allowed; box-shadow: none;`.
- Buttons must use `font-family: var(--font-machine)` (Space Grotesk) with weight 600.
